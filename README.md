# inialum-lp

Landing Page for INIALUM

## Development

> [!NOTE]
>
> - This project uses Astro. You can read the documentation [here](https://docs.astro.build).

### Setup

1. Clone this repository
2. Install dependencies

   ```shell
   pnpm install
   ```

3. Run the development server

   ```shell
   pnpm run dev
   ```
